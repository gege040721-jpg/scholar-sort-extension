# Google Scholar Citation Sorter

A Chrome extension that automatically sorts Google Scholar search results by citation count.

## Features

- Sort search results by citation count (highest first)
- Toggle between sorted and original order
- Visual citation badges on each result
- Works on all Google Scholar search pages

## Installation

### For All Platforms (Windows / Mac / Linux)

1. Download this repository:
   - Click the green **Code** button above
   - Select **Download ZIP**
   - Extract the ZIP file to a folder

2. Install in Chrome:
   - Open Chrome and go to `chrome://extensions/`
   - Enable **Developer mode** (toggle in top right corner)
   - Click **Load unpacked**
   - Select the extracted folder

3. Done! Go to [Google Scholar](https://scholar.google.com) and search for something.

## Usage

1. Go to Google Scholar and search for any topic
2. Click the blue **"Sort by Citations"** button above the results
3. Results will be reordered from highest to lowest citation count
4. Click the green **"Restore Original Order"** button to undo
